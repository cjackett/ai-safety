# Resources

Web documentation, blog posts, and online resources saved as markdown for offline reference.

## Naming Convention

Files follow the format: `<year>_<source>_<title>.md`

- **Year**: Publication or access year (4 digits)
- **Source**: Organisation or website (lowercase, e.g., anthropic, openai, etc.)
- **Title**: Descriptive title with dashes replacing spaces (lowercase)

**Examples:**
- `2024_anthropic_core-views-on-ai-safety.md`
- `2024_aisi_frontier-ai-evaluation-reports.md`
- `2024_openai_preparedness-framework.md`
