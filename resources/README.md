# Resources

Web documentation, blog posts, and online resources saved as markdown for offline reference.

## Naming Convention

Files follow the format: `<date>_<source>_<title>.md`

- **Date**: Publication date in `YYYY-MM-DD`
- **Source**: Organisation or website (lowercase, e.g., anthropic, openai, etc.)
- **Title**: Descriptive title with dashes replacing spaces (lowercase)

**Examples:**
- `2023-03-08_anthropic_core-views-on-ai-safety.md`
- `2024-01-15_aisi_frontier-ai-evaluation-reports.md`
- `2024-07-18_openai_preparedness-framework.md`
