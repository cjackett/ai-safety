# Papers

Academic papers and technical reports supporting AI safety evaluation research.

## Naming Convention

Files follow the format: `<year>_<author>_<title>.pdf`

- **Year**: Publication year (4 digits)
- **Author**: Last name or organisation (lowercase)
- **Title**: Descriptive title with dashes replacing spaces (lowercase)

**Examples:**
- `2023_shevlane_model-evaluation-for-extreme-risks.pdf`
- `2022_anthropic_constitutional-ai-harmlessness-from-ai-feedback.pdf`
- `2024_openai_gpt-5-2-system-card.pdf`
